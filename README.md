# raccoon5
 小浣熊漫画CMS5.0

# 程序特色
- 完善的会员系统，带支付系统，带VIP功能，带推广功能
- 完善的APP接口
- 完善的火车头API
- 阅读页可以选择一页几图，为网站增加PV
- 章节预加载，在阅读下一章时不需要再等待载入
- 三端并存：PC端，移动端，MIP端，MIP端可以跳转到移动端的阅读页，解决MIP不能放广告的问题（MIP做引流，移动端做真正变现）
- 作者系统
- 对接公众号系统
- 前台一些列表页面使用vue上拉加载，用户体验更好
